# Product Requirements Document (PRD)

## Project Name
AI Task Manager

## Overview
AI Task Manager is a web application that allows users to create, manage, and track tasks. 
The application provides a clean interface and stores tasks using a backend server.

## Goals
- Allow users to create tasks
- Allow users to mark tasks as completed
- Display active tasks
- Provide a structured backend and frontend architecture

## Target Users
Students and individuals who need a simple task management tool.

## Functional Requirements
- Add new task
- Display task list
- Mark task as completed
- Store tasks on server

## Technical Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Styling: Tailwind CSS
- Structure: Client/Server architecture

## Future Improvements
- User authentication
- Database persistence
- Task categories
- AI-based prioritization
